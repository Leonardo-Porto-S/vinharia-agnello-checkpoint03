Sobre o Projeto

Sistema de gerenciamento de estoque da Vinheria Agnello, desenvolvido em JavaScript com objetos, arrays e funções de alta ordem.

Toda a saída de dados é feita via console.log() e alert(). Abra o console do navegador (F12 → Console) para ver os resultados.

Funcionalidades

Listar Vinhos

Percorre o array e exibe todos os vinhos. forEach

Estoque Crítico

Filtra vinhos com menos de 5 unidades. filter

Estoque Total

Soma todas as unidades em estoque. reduce

Nomes em Caixa Alta

Transforma os nomes para maiúsculas. map

Adicionar Vinho

Insere um novo objeto no array. push

Como testar

1. Abra esta página no navegador.

2. Pressione F12 e acesse a aba Console.

3. Os resultados serão exibidos automaticamente.

4. Um alert() com o resumo geral também será apresentado.
