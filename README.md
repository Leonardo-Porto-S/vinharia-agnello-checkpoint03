# Vinheria Agnello – Checkpoint 03
 
Sistema de gerenciamento de estoque de vinhos desenvolvido com JavaScript puro, como parte do **Checkpoint 03** da disciplina de Desenvolvimento Web.
 
---
 
## 📝 Descrição
 
O projeto simula o controle de estoque da *Vinheria Agnello*, aplicando os conceitos de **objetos**, **arrays** e **funções de alta ordem** do JavaScript. Não há manipulação de DOM — todas as saídas são feitas via `console.log()` e `alert()`.
 
---
 
## ⚙️ Funcionalidades
 
| Função | Método utilizado | O que faz |
|---|---|---|
| `listarVinhos()` | `forEach` | Exibe todos os vinhos cadastrados no console |
| `vinhosBaixoEstoque()` | `filter` | Lista vinhos com menos de 5 unidades |
| `estoqueTotal()` | `reduce` | Calcula o total de unidades em estoque |
| `nomesEmCaixaAlta()` | `map` | Retorna os nomes convertidos para maiúsculas |
| `adicionarVinho(...)` | `push` | Insere um novo objeto no array de vinhos |
| `exibirResumo()` | — | Exibe um `alert()` com o resumo geral |
